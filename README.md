Este projeto foi feito para a disciplina de Fundamentos de Programação para Internet.  
A ideia é criar um **gerador de currículos** onde o usuário preenche seus dados e o sistema monta automaticamente um currículo formatado.
---
##  O que o sistema faz
- Permite preencher dados pessoais (nome, data de nascimento, etc)
- Calcula automaticamente a idade com JavaScript
- Adiciona novas experiências e cursos de forma dinâmica
- Gera o currículo usando PHP
- Permite imprimir ou salvar o currículo 
- Interface simples feita com HTML, CSS e Bootstrap
---
##  Tecnologias usadas
- PHP  
- HTML  
- CSS  
- JavaScript / jQuery  
- Bootstrap 5  
---
##  Como rodar o projeto

1. Instale o [XAMPP](https://www.apachefriends.org/pt_br/index.html)  
2. Coloque a pasta do projeto dentro de: C:\xampp\htdocs\curriculo
3. Inicie o Apache no painel do XAMPP  
4. No navegador, acesse: [http://localhost/curriculo/insert.php](http://localhost/curriculo/)
